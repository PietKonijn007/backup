How to Sync Files to AWS S3

Follow these steps to sync your Google Drive files to AWS S3.

1 Get File/Folder ID from Google Drive

For Files:

  1. Go to the Files page
  2. Right-click on a file → "Open in Google Drive" or hover to see the file ID in the browser
  3. The URL looks like: https://drive.google.com/file/d/FILE_ID/view
  4. Copy the FILE_ID part

For Folders:

  1. Click on a folder to navigate into it
  2. Check the browser URL: /?folder_id=FOLDER_ID
  3. Copy the FOLDER_ID

2 Sync Using Browser Console

Open your browser console (F12 or right-click → Inspect → Console) and run:

Sync a Single File:

fetch('/api/sync/file', {
    method: 'POST',
    headers: {'Content-Type': 'application/json'},
    body: JSON.stringify({file_id: 'YOUR_FILE_ID_HERE'})
})
.then(r => r.json())
.then(data => console.log('Result:', data))

Sync Entire Folder:

fetch('/api/sync/folder', {
    method: 'POST',
    headers: {'Content-Type': 'application/json'},
    body: JSON.stringify({folder_id: 'YOUR_FOLDER_ID_HERE'})
})
.then(r => r.json())
.then(data => console.log('Result:', data))

Sync Multiple Files:

fetch('/api/sync/files', {
    method: 'POST',
    headers: {'Content-Type': 'application/json'},
    body: JSON.stringify({file_ids: ['FILE_ID_1', 'FILE_ID_2', 'FILE_ID_3']})
})
.then(r => r.json())
.then(data => console.log('Result:', data))

3 Sync Using Terminal/API

Using curl:

# Sync single file
curl -X POST http://localhost:8080/api/sync/file \
  -H "Content-Type: application/json" \
  -d '{"file_id": "YOUR_FILE_ID"}'

# Sync folder
curl -X POST http://localhost:8080/api/sync/folder \
  -H "Content-Type: application/json" \
  -d '{"folder_id": "YOUR_FOLDER_ID"}'

4 Check Results in AWS S3

  1. Go to AWS S3 Console
  2. Open your bucket: hofkens-backup
  3. Look in the google-drive/ folder
  4. Your synced files will appear there!

Pro Tips:

  • Syncing happens in real-time - download from Drive, upload to S3
  • Files are temporarily stored in /tmp/backup-sync then deleted
  • Large files may take a few minutes to sync
  • Check the terminal logs for sync progress
  • Folder sync downloads ALL files in the folder recursively
