How It Works

The sync daemon continuously monitors your Google Drive for changes and automatically syncs them to AWS S3.

  • Automatic Mode: Daemon checks for new/modified files every 5 minutes (configurable)
  • Manual Sync: Browse files and select specific items to sync
  • Reliable: Automatic retries and error handling
  • Efficient: Only syncs new or modified files
